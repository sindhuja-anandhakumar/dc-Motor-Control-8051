# DC Motor Control using 8051

This project controls a DC motor using an AT89C51 microcontroller and L293D motor driver.

## Components
1.AT89C51
2.L293D
3.DC Motor
4.2 Push Buttons
5.Crystal Oscillator
6.Keil uVision
7.Proteus

## Folder Structure
code/
keil/
Proteus/
Images/

## How to Run
1. Open the Keil project.
2. Build the project to generate the HEX file.
3. Open the Proteus simulation.
4. Set the Program File to the generated HEX file.
5. Run the simulation.
